# lt-jshint - JSHint for Lighttable

Integrates [jshint](http://jshint.com/) linting in Lighttable.

Alternate implementation to the [JSHint](https://github.com/LightTable/JSHint) plugin, that uses the [Linters](https://github.com/bbbates/lt-lint) to display results.
